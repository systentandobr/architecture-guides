## Uso de Cores no Chatbot

O componente ChatbotCollector utiliza as cores de acordo com o domínio especificado:

- **Header**: Utiliza a cor principal do domínio
- **Mensagens do Assistente**: Utiliza um background mais claro da cor principal do domínio
- **Botões de Sugestão**: Utiliza a cor principal do domínio
- **Indicador de Progresso**: Utiliza a cor principal do domínio com variações de opacidade

## Tokens Tipográficos

O chatbot utiliza a seguinte tipografia:

- **Títulos**: Inter, 20px, Bold
- **Mensagens**: Inter, 16px, Regular
- **Input**: Inter, 16px, Regular
- **Prompts Sugeridos**: Inter, 14px, Medium

## Espaçamento e Layout

O componente de chatbot utiliza um sistema de espaçamento consistente:

- **Space XS**: 4px
- **Space S**: 8px
- **Space M**: 16px
- **Space L**: 24px
- **Space XL**: 32px

O layout interno do chatbot segue uma estrutura de grid flexível, permitindo adaptação a diferentes tamanhos de tela.

## Responsividade

O componente ChatbotCollector é totalmente responsivo:

- **Desktop**: Layout completo com sugestões em linha
- **Tablet**: Layout adaptativo com mensagens em largura proporcional
- **Mobile**: Layout vertical com sugestões empilhadas

## Acessibilidade

Para garantir a acessibilidade, as cores do chatbot respeitam um contraste mínimo de 4.5:1 para texto e elementos interativos. Além disso, todos os componentes possuem atributos ARIA apropriados para leitores de tela.
